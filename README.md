## рҹҺҜ About The Project

> **DoMitra** (meaning *"Home Friend"* вҖ” your trusted local helper) is a full-stack Local Service Booking Platform developed as an internship project at **Genplus Training and Consulting Services Pvt. Ltd. (Genesis Training)**.

The platform bridges the gap between **customers** who need home services and **local professionals** who provide them вҖ” making it easy to discover, book, and pay for trusted service providers in your area.

**Why DoMitra?**
- рҹ”Қ Finding reliable local service providers is a pain вҖ” DoMitra solves that
- рҹ“… No more phone tag вҖ” book your plumber, electrician, or tutor in clicks
- рҹ’і Secure, hassle-free payment workflows
- вӯҗ Role-based trust вҖ” verified providers, protected customers

## вңЁ Features

### рҹ”җ Authentication & Authorization
- **JWT-based Role Authentication** вҖ” Separate flows for Customers, Providers & Admins
- **Secure login/signup** with token-based session management
- **Role-based access control (RBAC)** for all API endpoints

### рҹӣҺпёҸ Service Management
- Browse & search local service providers by category and location
- Service listings with pricing, availability, and ratings
- Provider profile management with service area configuration

### рҹ“… Booking System
- Real-time booking with slot selection
- Booking status tracking: `PENDING вҶ’ CONFIRMED вҶ’ COMPLETED`
- Cancellation and reschedule support
- Customer booking history dashboard

### рҹ’і Payment Workflows
- Secure payment processing integration
- Invoice and receipt generation
- Payment status tracking per booking

### рҹ”§ Admin Panel APIs
- Manage all users, providers, and service categories
- Approve/reject service provider applications
- Platform-wide analytics endpoints

## рҹҡҖ Getting Started

### Prerequisites
```bash
Java 17+
Node.js 18+
MySQL 8+
Maven 3.8+
```

### 1пёҸвғЈ Clone the Repository
```bash
git clone https://github.com/prashB2D/DoMitra-Service-Booking.git
cd DoMitra-Service-Booking
```

### 2пёҸвғЈ Setup the Database
```sql
CREATE DATABASE domitra_db;
```

### 3пёҸвғЈ Configure Backend
```bash
cd backend
# Edit src/main/resources/application.properties
# Set your DB credentials and JWT secret
```

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/domitra_db
spring.datasource.username=YOUR_USERNAME
spring.datasource.password=YOUR_PASSWORD
jwt.secret=YOUR_JWT_SECRET
```

### 4пёҸвғЈ Run the Backend
```bash
mvn spring-boot:run
# Server starts on http://localhost:8080
```

### 5пёҸвғЈ Run the Frontend
```bash
cd frontend
npm install
npm run dev
# App starts on http://localhost:3000
```
